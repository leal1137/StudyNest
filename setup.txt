Kör fäljande i wsl:

sudo apt update
sudo apt install nodejs npm
Restart terminal
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash
Restart terminal
nvm install --lts
npm install express
npm install socket.io

Skriv node server.js för att starta servern
Öppna http://localhost:3000 i två private fönster.
Det man skriver kommer dycka upp i båda fönstrena.
